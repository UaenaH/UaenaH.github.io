## Technical Specifications

The power of a server, the ruggedness of a sensor.

| Feature | Specification |
| :--- | :--- |
| **CPU** | Quad-core ARM® Cortex®-A72 @ 1.8GHz |
| **Memory** | 8GB LPDDR4 ECC RAM \| 64GB eMMC 5.1 Storage |
| **Operating System** | LidarOS (Debian-based) / RTOS / Ubuntu Core |
| **Connectivity** | Dual GbE, 5G/LTE Cat.16, Wi-Fi 6, LoRaWAN |
| **I/O Ports** | 2x RS-485/232, 4x DI/DO, 2x USB 3.0, CAN 2.0B |

## Deploy in Minutes

1. **Connect**  
   Plug in your industrial sensors via RS-485 or Ethernet and power up the gateway.

2. **Configure**  
   Use our web-based dashboard or CLI to define data paths and security credentials.

3. **Deploy**  
   Push your containerized edge logic and start streaming data to your cloud of choice.
